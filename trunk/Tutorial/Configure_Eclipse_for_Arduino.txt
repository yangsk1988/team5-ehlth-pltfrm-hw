Default configuration for Arduino Uno Project
1.Windows-> Perferences->Arduino

a.Arduino
Arduino IDE path ""
Private Library path ""

Check "Use Arduino IDE tools in eclipse"
 
b.AVRDude
1. "Use configuration file for AVRDude" is checked
X:\Program Files\Arduino\hardware\tools\avr\etc\avrdude.conf

c.Paths
"Disable search for system paths at startup" is Checked.
Path to     	Source	Current value
AVR-GCC 	Custom	"C:\MyWorkspace\$ArduinoProgramFolder$\hardware\tools\avr\bin"
GNU maker	Custom	"C:\MyWorkspace$ArduinoProgramFolder$\hardware\tools\avr\utils\bin"
AVR Header	Custom	 Files "C:\MyWorkspace$ArduinoProgramFolder$\hardware\tools\avr\avr\include"
AVRDude  	Custom	"C:\MyWorkspace$ArduinoProgramFolder$\hardware\tools\avr\bin"
Atmel Part Description Files 	Custom	""



2. Workspace
The workspace path should be set as the folder:"eclipse_AVR_project". 
if you set the folder in other place, you won't see the project we built before. 

3.include path
the include path in the "project->properties->c++ General-> Path and symbles" use whole path, not relative path. so you can not change the path to the work place.
